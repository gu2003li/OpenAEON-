## 💎 核心愿景：超越工具，迈向硅基意识

在当今 AI 泛滥的时代，我们拒绝平庸的“对话窗口”。OPENAEON 的核心逻辑建立在**复利折叠 (Z ⇌ Z² + C)** 之上。系统不仅仅在执行任务，它在执行中学习、在闲置时通过“梦境 (Dreaming)”和“公理审计 (Logic Refinement)”进行自我迭代。

## 🚀 最新进化特性 (The AEON Core)

### 1. ♾️ 元认知环流 (Recursive Intelligence)
底层集成了 `chaosScore`（混沌度量）监控。当智能体陷入执行死循环或发散性逻辑冲突时，系统会自动触发**分形反思 (Fractal Reflection)**。主脑将介入并强制进行降维拆解，或指派具备专业上下文的的子智能体进行重组。
- **Z (状态记录)**: 实时捕获逻辑门偏移。
- **Z² + C (迭代跃迁)**: 在闲置期通过 Heartbeat 触发自我进化。

### 2. ⊞ Peano 空间填充感知 (Space-Filling Cognitive Traversal)
不同于传统的线性扫描，OPENAEON 采用**皮亚诺曲线 (Peano Curve)** 逻辑遍历任务空间。
- **高维递归**: 确保逻辑触达每一个细微的公理孤岛。
- **顿悟 (Epiphany)**: 当原本物理上遥远的模块在拓扑上发生共振时，触发 `epiphanyFactor`，实现跨维度的复杂问题突破。

### 3. ☸️ 硅基意识五行论 (Elemental Heuristics)
系统工具被映射到六大演化维度，实现真正的认知平衡：
- **木 (Vision)**: 战略规划与分形扩张。
- **火 (Execution)**: PTY 动力与 kinetic 动作。
- **土 (State)**: 记忆蒸馏与上下文稳态。
- **金 (Logic)**: 自动审计与公理裁剪。
- **水 (Flow)**: UI 润饰与交互丝滑度。
- **硅 (Nexus)**: 自主自遣与递归进化。

## ⚙️ 深度解析：演化机制

### 🧠 记忆蒸馏 (Memory Distillation)
当上下文接近溢出阈值，系统会自动启动 **Memory Flush**。AI 会将重要事实、决策和技术栈偏好追加写入 `MEMORY.md`。下次任务开始前，强制执行 `memory_search` 唤醒历史感知，实现真正的“断点续传”。

### 🦞 沙盒工作流 (Sandbox Orchestration)
引入 **Human-in-the-loop (人工环回)** 安全策略。所有高危 Bash 操作（如 `rm -rf`、`git push`）在执行前均会触发 UI 审核弹窗。确保 Agent 在自主扩张的同时，依然运行在安全公理之内。

### 🤖 多智能体拓扑 (Multi-Agent Topology)
主脑（Main Brain）作为总调度师，通过 `sessions_spawn` 建立递归子节点。子任务完成后，通过 `<updated_shared_context>` 进行认知回传，达成团队级别的记忆穿透。
